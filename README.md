# Features

**App is live at [tasks.oureapp.com](https://tasks.oureapp.com) !**

## 🚧 Version 1 (Alpha without backend) 🚧

-   ✔️ Create todolist
-   ✔️ Task priority
-   ✔️ Task limit date
-   ✔️ Sorting system
-   ✔️ Filtering system
-   ✔️ Reward message on task complete
-   ✔️ Filtering and sorting menus
-   ❌ Create task
-   ❌ Edit task
-   ✔️ Delete task
-   ❌ Create task list
-   ❌ Edit task list
-   ❌ Delete task list
